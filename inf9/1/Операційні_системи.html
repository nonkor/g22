Операційні системи: Тоді і зараз

Клікніть на скріншот або натисніть пробіл, щоб дізнатися більше про операційну систему
Ранні системи (1980-90-ті роки)
MS-DOS (Command Line)

MS-DOS - Командна операційна система

Період: 1981-1995 роки

  • Інтерфейс: CLI (тільки текстові команди)
  • Користувачі: Однокористувацька
  • Задачі: Однозадачна
  • Особливості: Основа для ранніх ПК, потребує знання команд
Mac OS Classic

Mac OS Classic - Перший популярний GUI

Період: 1984-2001 роки

  • Інтерфейс: GUI (перший масовий графічний інтерфейс)
  • Користувачі: Однокористувацька
  • Задачі: Спочатку однозадачна, пізніше кооперативна багатозадачність
  • Історичне значення: Популяризував миш, вікна, іконки
Windows 98

Windows 98 - Ера Інтернету

Період: 1998-2006 роки

  • Інтерфейс: GUI з інтегрованим веб-браузером
  • Користувачі: Багатокористувацька (обмежено)
  • Задачі: Багатозадачна
  • Нововведення: USB, DVD, інтеграція з Інтернетом
Сучасні настільні системи
Windows 11

Windows 11 - Найновіша Windows

Випуск: 2021 рік

  • Інтерфейс: GUI з центрованим меню Пуск
  • Користувачі: Багатокористувацька
  • Задачі: Багатозадачна з покращеною продуктивністю
  • Особливості: Microsoft Teams, віджети, покращена безпека
macOS

macOS - Система Apple

Сучасна версія: macOS Sequoia (2024)

  • Інтерфейс: GUI з Dock та меню
  • Користувачі: Багатокористувацька
  • Задачі: Багатозадачна з відмінною оптимізацією
  • Особливості: Unix-основа, інтеграція з iPhone/iPad
Ubuntu Linux

Ubuntu - Популярний Linux

Випуск: Кожні 6 місяців з 2004 року

  • Інтерфейс: GUI (GNOME) + CLI Terminal
  • Користувачі: Багатокористувацька
  • Задачі: Багатозадачна
  • Переваги: Безкоштовна, безпечна, відкритий код
Мобільні системи
iOS (iPhone/iPad)

iOS - Мобільна система Apple

Випуск: 2007 рік з iPhone

  • Інтерфейс: GUI (мультитач, жести)
  • Користувачі: Однокористувацька
  • Задачі: Багатозадачна (обмежена для економії батареї)
  • Особливості: App Store, висока безпека, закрита система
Android

Android - Найпопулярніша мобільна ОС

Випуск: 2008 рік від Google

  • Інтерфейс: GUI (сенсорний з віджетами)
  • Користувачі: Однокористувацька
  • Задачі: Багатозадачна
  • Особливості: Відкритий код, Google Play, налаштування
Спеціалізовані системи
Chrome OS

Chrome OS - Хмарна система Google

Випуск: 2011 рік для Chromebook

  • Інтерфейс: GUI (веб-орієнтований)
  • Користувачі: Багатокористувацька
  • Задачі: Багатозадачна (в браузері)
  • Особливості: Швидке завантаження, все в хмарі, безпека
Android TV

Android TV - Система для телевізорів

Випуск: 2014 рік від Google

  • Призначення: Вбудована ОС для Smart TV
  • Інтерфейс: GUI (пульт дистанційного керування)
  • Задачі: Багатозадачна (стрімінг, додатки)
  • Особливості: Google Play Store, голосові команди, Chromecast
watchOS

watchOS - Система для Apple Watch

Випуск: 2015 рік з Apple Watch

  • Призначення: Вбудована ОС для розумних годинників
  • Інтерфейс: GUI (сенсорний екран + Digital Crown)
  • Задачі: Обмежена багатозадачність (фітнес, повідомлення)
  • Особливості: Відстеження здоров'я, Siri, синхронізація з iPhone
