Розуміння програмного забезпечення та його різновидів

Основні поняття

Програмне забезпечення

Сукупність всіх програм, що працюють на комп'ютері. Включає операційну систему, додатки, утиліти та драйвери. Без програмного забезпечення комп'ютер - це просто набір електронних компонентів.

Операційна система

Головна програма комп'ютера, яка керує всіма його ресурсами, забезпечує взаємодію між користувачем і апаратним забезпеченням, та створює платформу для роботи інших програм.

Прикладне ПЗ

Програми, призначені для вирішення конкретних завдань користувача: текстові редактори, браузери, ігри, графічні редактори, медіаплеєри тощо.

Системне ПЗ

Призначення: керування роботою комп'ютера та його ресурсами

  • Операційні системи - Windows, macOS, Linux
  • Драйвери - програми для роботи з пристроями
  • Утиліти - антивіруси, архіватори, очищувачі диска
  • Прошивки - програми в пам'яті пристроїв

Прикладне ПЗ

Призначення: вирішення конкретних завдань користувача

  • Офісні програми - Word, Excel, PowerPoint
  • Браузери - Chrome, Firefox, Safari
  • Медіаплеєри - VLC, Spotify, YouTube
  • Ігри - комп'ютерні та мобільні ігри
  • Графічні редактори - Photoshop, GIMP

Інструментальне ПЗ

Призначення: створення, налагодження та супровід програм

  • Текстові редактори коду - Visual Studio Code
  • Компілятори - перетворюють код у виконувані файли
  • Відладчики - знаходять помилки в програмах
  • Системи контролю версій - Git

Що таке операційна система?

Функції операційної системи

Керування ресурсами
  • Розподіл процесорного часу
  • Управління пам'яттю
  • Робота з файловою системою
Інтерфейс користувача
  • Графічний інтерфейс (GUI)
  • Командна оболонка
  • Меню та вікна
Забезпечення безпеки
  • Контроль доступу
  • Захист файлів
  • Управління правами користувачів

Операційна система як посередник

ОС працює як "перекладач" між користувачем і комп'ютером. Коли ви клікаєте на файл, ОС знає, яку програму запустити. Коли програма потребує пам'ять, ОС її виділяє. Без ОС програми не змогли б працювати разом.

Хронологія розвитку ОС

1940-50-ті
Перші комп'ютери без ОС. Програмування через перемикачі та картки.

Перші ОС для мейнфреймів. Пакетна обробка завдань.

1980-ті

1990-ті
Windows 95 - перша популярна ОС з графічним інтерфейсом для ПК.
2000-ні
Mac OS X, Windows XP. Стабільні системи для домашнього використання.
2010-ні
Мобільні ОС: iOS, Android. Хмарні сервіси.

Різновиди операційних систем

За призначенням

ПК (персональні комп'ютери):
  • Windows - найпопулярніша для домашнього використання
  • macOS - для комп'ютерів Apple
  • Linux - для програмістів та серверів
Мобільні пристрої:
  • Android - найпоширеніша мобільна ОС
  • iOS - для iPhone та iPad
Вбудовані системи:
  • Smart TV - в телевізорах Samsung, LG
  • Маршрутизатори та роутери
  • Автомобільні системи
  • Побутова техніка (холодильники, пральні машини)

За інтерфейсом

Графічні (GUI - Graphical User Interface):
  • Вікна, кнопки, меню, іконки
  • Керування мишею та дотиками
  • Зручно для повсякденного використання
  • Приклади: Windows, macOS, Android
Командного рядка (CLI - Command Line Interface):
  • Текстові команди
  • Швидше для досвідчених користувачів
  • Менше ресурсів комп'ютера
  • Приклади: MS-DOS, Linux Terminal

За кількістю користувачів

Однокористувацькі:
  • Тільки один користувач за раз
  • Простіша архітектура
  • Приклади: старі версії Windows, MS-DOS
  • Більшість мобільних ОС
Багатокористувацькі:
  • Кілька користувачів одночасно
  • Розподіл ресурсів між користувачами
  • Система прав доступу
  • Приклади: Linux, Windows Server, macOS

За підтримкою задач

Однозадачні:
  • Виконує тільки одну програму за раз
  • Простіше управління ресурсами
  • Приклади: MS-DOS, ранні ОС
  • Деякі вбудовані системи
Багатозадачні:
  • Кілька програм працюють одночасно
  • Розподіл процесорного часу
  • Перемикання між завданнями
  • Приклади: всі сучасні ОС

Файлові системи

Що таке файлова система?

Спосіб організації та зберігання файлів на носії інформації. Визначає, як дані записуються, читаються та організовуються в папки.

  • NTFS - файлова система Windows
  • APFS - сучасна файлова система macOS
  • ext4 - популярна в Linux
  • FAT32 - для флешок та сумісності

Особливості файлових систем

  • Максимальний розмір файлу
    • FAT32 - до 4 ГБ
    • NTFS - до 16 ТБ
    • APFS - практично необмежено
  • Журналювання - відновлення після збоїв
  • Права доступу - хто може читати/писати
  • Шифрування - захист даних

Важливо знати

Вибір операційної системи залежить від ваших потреб: Windows - для ігор та широкого вибору програм, macOS - для творчої роботи та безпеки, Linux - для програмування та серверів. Кожна система має свої переваги!
